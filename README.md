# AI Interview Prep

A full-stack web application that uses Gemini API with structured output to generate personalized interview preparation reports from your resume and job description.

## Features

- Upload resume (PDF) or write a self description
- Paste any job description
- AI generates a personalized report with:
  - Technical questions with model answers
  - Behavioral questions with STAR method answers
  - Skill gap analysis
  - 7-day preparation roadmap
- View all past reports on the home page

## Tech Stack

**Frontend**
- React.js
- React Router
- Axios
- SCSS

**Backend**
- Node.js
- Express.js
- MongoDB + Mongoose
- Google Gemini API
- Multer (file uploads)
- JWT Authentication

### Prerequisites
- Node.js
- MongoDB
- Google Gemini API key
